# Project Pelanggaran Kode

Website pelaporan pelanggaran kode etik pemasaran produk pengganti ASI.

## Tech Stack
- React.js (Vite)
- Tailwind CSS
- Leaflet (Maps)
- Supabase (Backend Integration ready)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup Environment Variables:
   Copy `.env.example` to `.env` and fill in your Supabase credentials.

3. Run Development Server:
   ```bash
   npm run dev
   ```

4. Build for Production:
   ```bash
   npm run build
   ```

## Features
- **Home**: Dashboard with statistics and interactive map.
- **Report**: Form to submit violations with location and evidence upload.
- **Responsive**: Mobile-friendly design (Android compatible).

## Deployment
Ready for deployment on Vercel. configuration `vercel.json` is included.
